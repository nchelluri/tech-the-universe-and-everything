by Narsa Chelluri, for Halihax

Yet, some predictions for the tech industry

  • Because being alive today means using tech,
  • Or having it used on your behalf.

Mo' tech, mo' problems

Sometimes, it's pretty serious

Though in this case, it wasn't as bad as it seemed

So what do I predict?

  • Software stack updates - updates as far as the eye can see
  • Sane default security settings across the board
  • Site Reliability Engineer becomes a househould term
  • Software security education will become mainstream for anyone working in tech
  • Tech professionals may be held legally responsible for ensuring the security of their systems

Software stack updates

  • The days of one-and-done software projects are coming to an end, if they aren't already over
  • Software-as-a-Service will continue to grow rapidly - because many forms of updates are already built-in
  • Every component of every stack will have auto-update functionality built-in, and it'll be unable to be shut off
  • Mistakes will be made. But we'll learn from them.

Sane default security settings

  • No more default passwords - browsing Shodan will no longer be a blackhat tradition
  • Some jurisdictions already mandate that software cannot be sold with default passwords -- each instance has its own credentials
  • State-of-the-art password hashing and encryption and security hygiene will be required by law

SREs everywhere!

  • Site Reliability Engineers keep software and firmware up-to-date, run security scans, scale systems, and generally keep operations running smoothly
  • SREs will grow in number
  • No organization that a runs tech stack will be without them
  • We'll probably see a host of SRE-as-a-Service companies for smaller organizations

Security education becomes mainstream for tech professionals

  • Everyone will have to learn how to not fall for a phishing attack
  • Web developers will know the OWASP Top 10 like the back of their hand
  • High quality tech security education will become widely available and required for most tech jobs

There may be legal consequences for negligence

  • Some ransomware can be stopped by anti-phishing knowledge
  • In the case of the hospital in Germany, the vulnerability was known to be in Citrix in January '20 but was exploited in September
  • With organizations facing catastrophic consequences due to what I loosely term negligence, the landscape is beginning to change

Conclusion

  • It's time for the industry to mature
  • Our lives all depend on it
  • That is my talk! Thank you for watching.
